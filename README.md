# Ace-Ifa-Boru

A premium Exam Preparation Platform built as a Telegram Mini App (TMA) with:
- **Frontend:** Next.js 14/15 with TypeScript & Tailwind CSS
- **Backend:** Node.js with Express/NestJS
- **Database:** Supabase (PostgreSQL)
- **Bot Framework:** grammY for push notifications

## Project Structure

```
├── packages/
│   ├── backend/          # Node.js API server
│   ├── frontend/         # Next.js TMA
│   └── shared/           # Shared types & utilities
├── database/             # SQL migrations & schemas
└── docs/                 # Documentation
```
